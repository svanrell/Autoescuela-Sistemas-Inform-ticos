# Product Context: Autoescuela de Sistemas

## Propósito
El objetivo de este proyecto es crear una plataforma educativa interactiva ("Autoescuela") para que los estudiantes de Sistemas Informáticos puedan practicar y evaluar sus conocimientos sobre hardware, comandos de Windows/Linux, redes IP y contenedores.

## Problema que resuelve
Los estudiantes a menudo necesitan una forma dinámica de repasar conceptos teóricos y comandos prácticos. Esta herramienta proporciona un entorno de test que simula un examen real con retroalimentación inmediata.

## Usuarios Objetivo
- Estudiantes de Formación Profesional de Grado Medio/Superior (Sistemas Informáticos).
- Opositores o entusiastas que quieran validar sus conocimientos técnicos.

## Funcionalidades Principales
- **Banco de Preguntas Dinámico**: Carga de preguntas desde un archivo JSON.
- **Test Interactivo**: Selección de respuestas con validación visual.
- **Barra de Progreso**: Visualización del avance durante el test.
- **Resultados Detallados**: Resumen de aciertos y fallos al finalizar.
- **Diseño Responsive**: Accesible desde ordenadores y dispositivos móviles.
