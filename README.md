# Autoescuela de Sistemas Pro 🚀

Plataforma educativa interactiva para estudiantes de sistemas informáticos.

## Características
- **6 Categorías Técnicas**: Hardware, Linux, Redes, Windows, Seguridad y Cloud.
- **Multilingüe**: Soporte total para Castellano, Català e English.
- **Ranking Global**: Leaderboard interactivo de estudiantes.
- **Historial Personal**: Seguimiento de tus puntuaciones pasadas.
- **Navegación Avanzada**: Botones de retroceso y avance en los exámenes.
- **Diseño Glassmorphism**: Interfaz moderna, responsiva y premium.

## Estructura
- `src/app.js`: Lógica de estado y orquestación.
- `src/components.js`: Generación de componentes HTML puros.
- `src/style.css`: Sistema de diseño y animaciones.
- `src/data*.json`: Bancos de preguntas multilingües.
