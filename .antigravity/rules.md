Reglas de Desarrollo:

Arquitectura: Seguir estrictamente el patrón MVC.

Modularidad: Todo el código debe estar dentro de funciones.

Funciones Puras: Las funciones no deben acceder a variables externas; todo debe pasar por parámetros.

Interfaz: El index.html es minimalista. Los componentes se generan en app.js mediante funciones.

Punto de Entrada: La función main() debe actuar como un menú o despachador de operaciones.
